# qv2ray-deb

copplation for debian10/Deepin20/UBUNTU20.04/LinuxMint20.1 etc.
test for debian10

## usage

download qv2ray deb package.

unzip v2ray-core to ~/.config/qv2ray/vcore

click .desktop to launch qv2ray or use 'qv2ray' command line
